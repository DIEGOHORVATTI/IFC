#4 - Classifique cada função de Reais em Reais, em função afim, linear, constante ou identidade.

	a) f(x) = 3/2x - 5:
		Essa função é afim, uma vez que possui a fórmula geral de uma equação de primeiro grau: y = ax + b. Nesse caso, temos: a = 3/2 e b = -5.
	
	b) f(x) = x:
		Essa função se refere a função identidade, onde temos os mesmos valores de f(x) para todo x que utilizamos.
	
	c) f(x) = - 3x:
		Nesse caso, temos uma função linear, que é um caso específico das funções afim.
		Nessa função, temos a fórmula geral "y = ax + b", onde b = 0.
	
	d) f(x) = - 5:
		Essa função é constante, pois para qualquer valor de x, f(x) vai ser sempre o mesmo.
	
	e) f(x)= 15 - 4/5x:
		Essa função também é afim, pelo mesmo motivo do item a). Nesse caso, temos: a = - 4/5 e b = 15.
	
	f) f(x) = - x:
	Essa função se refere ao contrário da função identidade, onde temos os valores de
	f(x) só que negativos para todo -x que utilizamos.
