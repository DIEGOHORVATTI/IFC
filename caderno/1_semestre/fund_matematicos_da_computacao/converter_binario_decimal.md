  Converta de binario para decimal
</h2>

<pre>
  a)

    (1 0 1 0 1)²
    | | | | | 
    | | | | -> 1*2⁰ =               = 1 
    | | | ---> 1*2¹ = (1)*(2*1)     = 2
    | | -----> 1*2² = (1)*(2*2)     = 4
    | -------> 1*2³ = (1)*(2*2*2)   = 8
    ---------> 1*2⁴ = (1)*(2*2*2*2) = 16
                                    ______
                                      3857

   (7 4 2 1)⁸
    | | | |  
    | | | -> 1*8⁰ =             = 1
